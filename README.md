# GNU/LINUX Dotfiles

In this repository are my personal arch/antergos dotfiles used on my laptops
(Thinkpad x240 and HP EliteBook 8470p) and desktop. Terminal I use is st which
is hardcoded to i3 config.

+ i3-gaps
+ vim
+ polybar
+ bash
+ fish
+ rofi
+ mpd
+ ncmpcpp
+ compton
+ ranger
+ zathura
+ sxiv
+ dunst
+ firefox (chrome only)
+ Xresources
